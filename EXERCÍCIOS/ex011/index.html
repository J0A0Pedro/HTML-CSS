Minha NOTA após a atividade:

A tag img será a imagem fixa que o site irá abrir normalmente e, de acordo com o tamamho da tela que do aparelho que usa o site, ele irá usar um dos outros dois sources para melhorar o desempenho do usuário. Se o usuário estiver conectando de um aparelho celular, o source de 750px será aberto no site ao invés do img fixo. Se o usuário estiver acessando de um aparelho com uma tela um poucom maior a que de um celular, como um tablet, o source de 1050px será aquele aberto. A imagem padrão sempre será aquele que está na tag img, mas os dois sources (750px e 1050 px) se adequam para melhor visão
