Principais Formatações

Negrito / ou Destaque

Nesta frase, temos um Termo em Negrito usando a tag <b> (Não é semântica. Ela não tem significado, ela só tem forma, estética.)

Agora, nesta frase, temos um Termo em Destaque usando a tag <strong> (Ela é Semântica.)

Itálico / ou Ênfase

Nesta frase, temos um Termo em Itálico usandoa tag <i> (Não é Semântica)

Nesta frase, temos um Ênfase em Intálico usando a tag <em> (Ela é Semântica.)

Teexto marcado

Também podemos criar um texto marcado usando a tag <mark>

Aqui, temos outro texto marcado no final colocando o style dentro do <mark> e selecionando background-color

Texto Grande e Pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo. Utilizando a tag <big> e a tag<small> respectivamente. (Semântica, mas a tag big já virou obsoleta e sairá de uso, então é recomendável não usá-la)

Texto Deletado.

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado. Usamos a tag <del> (Semântica)

Texto inserido.

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. Aqui, usamos a tag <ins> (Semântica)

Existe também o sublinhado, semelhante ao inserido, mas o subilhado é uma tag NÃO-semântica e é puramente visual. Usamos a tag <u>

Texto Sobrescrito.

Para inserir coisa do tipo x20 + 3. Usamos aqui a tag <sup> no número 20 para elevá-lo.

Texto Subscrito.

Para inserir coisas do tipo H2O. Usamos aqui a tag <sub> no número 2
